# Cafe Lumiere - Web制作サンプル

## 🌐 デモサイト
https://630masarouta.github.io/cafe-lumiere/

---

## 📌 概要
架空のカフェ「Cafe Lumiere」の公式サイトを想定した
静的Webサイト制作サンプルです。

小規模店舗向けのシンプルなコーポレートサイトを想定し、
実務案件を意識して制作しています。

---

## 🎯 現在対応可能範囲
- 静的ホームページ制作（HTML / CSS）
- レスポンシブ対応（スマートフォン最適化）
- テキスト・価格変更
- メニュー追加
- セクション追加
- 軽微なデザイン修正
- 既存HTML/CSSの修正対応

---

## 💻 使用技術
- HTML5
- CSS3
- レスポンシブ設計（768pxブレークポイント）
- Gitによるバージョン管理
- GitHub Pagesを用いた静的サイト公開経験

---

## ✨ 実装ポイント
- セマンティックHTML構造
- モバイルファースト設計
- アクセシビリティ配慮（alt属性・focus対応）
- 保守性を意識したシンプル構造
